argument_count


Description

This read-only variable returns the number of "arguments" (parameters) that are passed through to a scripted function. This is used in conjunction with the built-in argument array to permit varying input arguments for a given function. The argument array has the format argument[0...n], where "n" is the last variable argument being passed into the function. Note that you can have a mix of fixed arguments and variable arguments in a function (as shown in the example below), although the first arguments passed in must always be the fixed ones.


Syntax:

argument_count;


Returns:

Integer


Example:

/// @function                 create_random(object, layer);
/// @param {int}    object    The object to create an instance of
/// @param {int}    layer     OPTIONAL! The layer to create it on
/// @description              Create an instance of the given object at a random position on the current layer or on the (optional) given layer

function create_random(_obj)
    {
    var _layer = layer;
    if argument_count > 1
        {
        _layer = argument[1];
        }
    var _x = irandom(room_width);
    var _y = irandom(room_height);
    instance_create_layer(_x, _y, _layer, _obj);
    }

The above code uses the argument_count variable to check if a layer ID has been passed into the function or not..
