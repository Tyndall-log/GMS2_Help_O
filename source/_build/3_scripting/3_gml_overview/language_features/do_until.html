Do / Until


A do statement is another way of iterating over one or more statements, and is really a "do... until" statement as you cannot have one without the other since you are telling GameMaker Studio 2 to do something until a specific expression returns true. It has this form:

do {<statement>} until (<expression>)


The statement (which can be a code block of multiple statements within curly brackets {}) is executed until the expression is found to be true, and the initial statement is always executed at least once. Be careful with your do loops, as you can easily make them loop forever, in which case your game will hang and not react to any user input anymore and they will have to force close the game. Below you can find an example of a typical way to use do... until:

do
    {
    x = random(room_width);
    y = random(room_height);
    }
until (place_free(x, y));


The above program tries to place the current object at a free position and will set the x/y variables as many times as required until the place_free() expression returns true.

For more examples of loop keywords please see the sections on repeat, while, and for.
